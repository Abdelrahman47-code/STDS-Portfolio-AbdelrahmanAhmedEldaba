# STDS Portfolio Projects

Welcome to the **STDS Portfolio** collection. This directory contains four distinct data analysis and business intelligence projects, showcasing a diverse range of tools and techniques including **SQL**, **Power BI**, **Tableau**, and **Python**.

Each project is designed to demonstrate proficiency in data extraction, processing, visualization, and actionable insight generation.

---

## üìÇ Project Highlights

### 1. [AdventureWorks Database Analysis Project](./AdventureWorks_SQL_Dashboard-main)
**A Comprehensive Business Intelligence Dashboard**

This project performs a deep-dive analysis of the classic AdventureWorks database (2011-2014). It features interactive dashboards that provide actionable insights into sales performance, product analytics, and operational metrics.

*   **Technology Stack:** Microsoft SQL Server, Microsoft Excel.
*   **Key Features:**
    *   Sales overview and territory insights.
    *   Deep-dive product analytics and logistic cost distribution.
    *   Employee performance tracking and category concentration analysis.
    *   Extensive data preprocessing using SQL (Data cleaning, merging, and view creation).

### 2. [Northwind Traders Performance Dashboard](./Northwind_Traders_Dashboard-main)
**Interactive Power BI Report**

A professional dashboard built using the Northwind Traders sample database. It offers a clear view of sales performance, order trends, and supply chain metrics for a fictional specialty foods company.

*   **Technology Stack:** Microsoft Power BI, Microsoft Excel.
*   **Key Features:**
    *   Dynamic filtering by Country, Shipper, and Year.
    *   KPI cards for quick performance assessment.
    *   Detailed analysis of Categories, Shippers, and Employees.
    *   Geographic distribution metrics.

### 3. [Super Store Sales Insights](./Super_Store_Sales_Dashboard-main)
**Advanced Tableau Dashboard**

A polished Tableau dashboard built on the Superstore dataset (2014-2017). This project focuses on interactive sales and profit analysis across different regions and customer segments.

*   **Technology Stack:** Tableau Desktop.
*   **Key Features:**
    *   High-level trends and geographic sales distribution.
    *   Deep dive into regional performance and sub-category metrics.
    *   Interactive filters for Region and Year.
    *   Complex dashboard actions and synchronized visuals.

### 4. [Used Cars in Egypt Analysis](./hatla2ee_used_cars_dashboard)
**Web Scraping & Exploratory Data Analysis**

This project involves the end-to-end process of scraping, cleaning, and analyzing a dataset of over 33,500 used car listings from the Egyptian market (Hatla2ee).

*   **Technology Stack:** Python (Selenium, BeautifulSoup, Pandas, Matplotlib, Plotly).
*   **Key Features:**
    *   Custom web scraper implementation.
    *   Exploratory Data Analysis (EDA) on price, mileage, and features.
    *   Insights into market trends, top brands, and monthly price evolution.
    *   Data cleaning and preprocessing workflows.
